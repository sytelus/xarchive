The goal is to develop a new Chrome browser extension called xarchive that reliably and robustly downloads all bookmarks from x.com for a logged in user. The reliability here means that downloaded bookmarks are complete and not partial. If needed, this extension would implement strategies to work around quirks of x.com as well as Chrome to ensure reliability. The extension will attempt to download as much information as possible for each bookmark, including the list of folders that were assigned to the each bookmark. All the downloaded information should be saved in json format.
